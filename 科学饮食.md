饮食与困倦
问题：什么样的饮食有助于改善困倦的情况？
警惕单一归因谬误：困倦的情况是由多方面的因素造成的，包括睡眠质量、长期压力或甲状腺功能、不只是饮食造成的。
加法思维偏见：在大多数时候，困倦时身体由于过度摄入而产生的负担。为了改善困倦，不该吃什么相对于该吃什么是一个更应该关注的问题。

饮食导致困倦的本质
餐后嗜睡(Postprandial Somnolence)叠加血糖震荡(Glycemic Variability)
核心机制
摄入特定食物后，副交感神经系统(休息与消化模式)过度激活，同时胰岛素激增导致随后的反应性低血糖，使得大脑的葡萄糖供应变得不稳定。
==什么样的食物，什么是副交感神经系统，过度激活的含义是什么，大脑的葡萄糖供应不稳定会导致什么后果==

困倦的原因
直接原因
大脑中负责觉醒的神经递质食欲素(Orexin)被抑制了，它是让人保持清醒和活跃的开关。
结构原因
为什么食欲素被抑制：血糖升高会直接抑制食欲素的分泌；摄入大量精制贪睡时，血糖飙升，食欲素断电。
第一性原理
能量守恒与分配：消化是一项极其耗能的代谢活动；当身体接收了大量由于高GI(升糖指数)事物产生的速燃燃料时，身体判定当前的最高优先级是储存能量而非狩猎或思考；胰岛素激增，血液涌向内脏，大脑收到关机休眠以利于储存的指令。

# 饮食结构
## 原则
多样化+适量+原型食物为主+稳定能量供给
## 组成
### 碳水化合物-基础燃料
原理
碳水的作用：碳水=>葡萄糖=>血液=>血糖上升
影响：胰岛素分泌
葡萄糖的去向
- 大脑
- 肌肉 (糖原)
- 多余的成为脂肪

重要的不是血糖升不升，而是升得快不快、稳不稳、降得急不急
问题链条
```
精制碳水 => 血糖快速飙升
=> 胰岛素大量分泌
=> 血糖迅速下降
=> 饿、困、烦、想再吃
```
控制碳水的关键在于控制血糖波动，而不是绝对量

不同碳水的本质差异
1). 分子结构：消化的速度
2). 膳食纤维：吸收速度
3). 食物结构：胃排空速度
4). 加工程度：血糖反应

精制碳水的问题
什么是精制碳水？
看食物结构被破坏到什么程度
- 谷物被去掉麸皮和胚芽
- 几乎只剩下淀粉，少了膳食纤维、矿物质、抗性淀粉
- 消化快
例子
- 白米饭、白面条
- 白面包、蛋糕
- 饮料、糖果
特点
- 结构被破坏
- 几乎不用消化
- 进入血液快
- 适合体力消耗后的快速补充
血糖反应
- 血糖迅速飙升
- 胰岛素大量分泌
- 随后血糖快速下降
==为什么这样过山车的血糖变化会使得人犯困、很快又饿和想吃甜食==

全谷物/复合碳水
例子
- 糙米、燕麦
- 全麦、藜麦
- 红薯、土豆
特点
- 含纤维
- 需要咀嚼
- 消化慢
- 血糖曲线平缓

控制碳水的方式
1). 控制形式：少吃精制碳水
2). 控制时机：白天多、晚上少
3). 控制搭配：碳水+蛋白质+脂肪+纤维
量化方法：一餐碳水大概是拳头大小的量


推荐来源
- 全谷物：糙米、藜麦、全麦面包
- 杂粮：藜麦、小米、玉米
- 薯类：红薯、土豆 (替代部分主食)
尽量少
- 精制糖、甜饮料精制碳水的
- 白面包、蛋糕、饼干
判断标准
	**血糖平稳、不犯困、不暴食**

蛋白质-维持肌肉、免疫、激素的核心
推荐来源
- 动物：鱼、鸡蛋、禽肉、瘦红肉
- 植物：豆腐、豆类、坚果
- 乳制品：牛奶、酸奶、奶酪
建议
- 每餐都应该有蛋白质
- 运动量大的人多摄入

脂肪
优质脂肪
- 橄榄油、菜籽油
- 坚果、牛油果
- 深海鱼 (Omega-3)
少吃
- 反式脂肪
- 过量油炸食品
判断标砖
	**吃完不腻、不反胃、不长期体重飙升

# 生酮饮食
人体的两套功能系统
1). 葡萄糖供能
特点
- 碳水 => 葡萄糖 => 血糖
- 快速、灵活
- 容易波动
适合
- 高强度运动
- 快速反应
2). 脂肪/酮体供能
特点
- 脂肪 => 肝脏 => 酮体
- 稳定、持续
- 依赖肝脏和线粒体
适合
- 长时间专注
- 稳定精神状态
**生酮的核心不是多脂肪，而是低胰岛素状态**

生酮让人觉得头脑清醒的原因
1). 血糖波动被强力压平
- 碳水极低 => 血糖几乎不波动
- 大脑不再经历高低摆动
2). 酮体是更干净的燃料
- 酮体代谢产生的ROS较低
- 对神经元更友好
- 神经炎症可能下降
==这里的神经元和神经炎症具体指的是什么，还有ROS==
3). 食欲信号被削弱
- 胰岛素低，饥饿素波动小
- 情绪不再被想吃东西牵着走
生酮的代价
1). 对甲状腺和神经系统的潜在压力
- 长期低碳，T3激素可能下降
- 情绪平直、动力下降
2). 对高强度运动和爆发力不友好
- 糖原储备低
- 无氧/变速能力下降
3). 微量元素和肠道风险
- 膳食纤维容易不足
- 钾、镁、钠失衡导致头痛、疲劳
4). 生酮不等于代谢更高级
