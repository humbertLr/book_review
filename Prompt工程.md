# 清晰目标和没有清晰思路之间的矛盾
## 原则
不要让AI给答案，而是让AI帮助构建思考框架
## 核心方法
1. 帮助拆解问题
2. 提供多个方案
3. 逐步迭代
## 第一层：问题探索
1. 拆解成几个关键步骤
2. 每个步骤需要哪些知识或方法
3. 哪些部分是最关键或最困难的
4. 完成这个任务通常有哪些常见方法
## 第二层：方案生成
提出**多种**解决方案，说明基本思路和需要的工具及方法，优缺点
## 第三层：执行规则
基于选定的方案具体设计流程：详细步骤，每一步需要的工具，可能遇到的问题，如何验证结果
## 技巧
1. 新的思考维度：可能忽略了哪些问题；如果是领域专家会如何思考这个问题。
2. 让AI做问题设计者：提出值得研究的问题，解释重要性和研究价值

# Meta prompt
本质：用一个prompt生成更好的prompt。
Meta prompt -- 生成Task Prompt -- 执行任务
## 基本结构
- 任务目标
- 背景信息
- 输出要求
- 优化目标
## 模板
```
作为一个高质量Prompt，请帮我设计一个高质量Prompt，用于完成以下任务：
任务：
[描述任务]
背景：
[相关信息]
要求：
1. Prompt应结构清晰
2. 包含思考步骤
3. 指定输出格式
4. 能得到深入分析
设计时请考虑：  
5. 角色设定  
6. 思考步骤  
7. 输出结构  
8. 结果质量
请输出：
1 最优Prompt  
2 Prompt结构说明  
3 如果需要可提供改进版本
```
